## Shaun's Business Notes App

## Assignment
This readme generator fulfills the requirements for the week 11 challenge. The assignment is to create an app that allows users to keep a running track of notes for themselves. The user should be able to add new notes and view old, saved notes.

## Purpose
To practice node application and different api requests while connecting them to a pre-built front-end.

## Built With
* Node
* Inquirer
* File System
* Express
* Heroku
